# Veterinary & Animal Care API Client and Request Builder VS Code Extension

> Build, test, and document API requests without leaving your editor. Postman alternative that lives in VS Code.

## Features

- Core functionality

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.d5872e19`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Veterinary & Animal Care API Client and Request Builder VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `d5872e19.enabled` | `true` | Enable/disable the extension |
| `d5872e19.autoRun` | `false` | Auto-run on file open |
| `d5872e19.logLevel` | `"info"` | Log verbosity |

## Requirements

- VS Code 1.74.0 or higher
